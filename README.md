# Personal Website

Firebase-hosted personal website.

## Stack
- Frontend: React + Vite

## Project structure
```
frontend/    # React app
backend/     # FastAPI service
infra/       # Terraform
```

## Local development
Frontend:
```
cd frontend
npm install
npm run dev
```


## Deployment
- Build frontend: `npm run build`
- Deploy to Firebase: `firebase deploy`

## Customize content
Edit `frontend/src/App.jsx` to update sections, projects, and contact details.
